Вывести строки таблицы лечений (TREAT.*), 
для которых сумма, начисленная на пациента, 
(TREAT.AMOUNTCL) является максимальной в рамках этого пациента

SELECT TREAT.*
// Выбираем все столбцы из таблицы TREAT.

FROM TREAT
// Указываем, что основной таблицей для выборки является TREAT.

JOIN (
    SELECT PCODE, MAX(AMOUNTCL) as MaxAmount
// Вычисляем максимальное значение AMOUNTCL для каждого пациента. MAX() 

    FROM TREAT
// Говорим, что мы берём данные из таблицы TREAT.

    GROUP BY PCODE
// Группируем все строки по PCODE.
// Это позволяет функции MAX() работать по каждой группе пациентов отдельно, 
// возвращая максимальное значение AMOUNTCL для каждой группы.

) as MaxTREAT
// Выполняем подзапрос, которому даём алиас MaxTREAT.

ON TREAT.PCODE = MaxTREAT.PCODE AND TREAT.AMOUNTCL = MaxTREAT.MaxAmount;
// TREAT.PCODE = MaxTREAT.PCODE - cвязываем обе таблицы по PCODE, чтобы объединить 
// данные по каждому пациенту.

// TREAT.AMOUNTCL = MaxTREAT.MaxAmount - мы также проверяем, что сумма начислений 
// (AMOUNTCL) в таблице TREAT совпадает с максимальной суммой (MaxAmount) из подзапроса.
// Таким образом, для каждого совпадения по PCODE и AMOUNTCL, мы получаем только 
// те записи из TREAT, где сумма начислений максимальна среди всех записей этого пациента.
