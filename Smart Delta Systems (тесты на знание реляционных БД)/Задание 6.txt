Вычислить вторую максимальную дату рождения в таблице пациентов, не используя 
агрегатных функций (по условию не может быть пациентов с одинаковой датой рождения)

SELECT BDATE
FROM CLIENTS
// Выбираем все записи поля BDATE (даты рождения) из таблицы CLIENTS.

ORDER BY BDATE DESC
// Дальше делаем сортировку в порядке убывания, 
// от самой поздней к самой ранней.
// Таким образом, самая последняя дата рождения будет первой 
// в результате выборки.

OFFSET 1 ROW
// Пропускаем самую первую строку, которая соответствует
// самой поздней дате рождения.

FETCH NEXT 1 ROW ONLY;
// Cколько строк необходимо вернуть после применения OFFSET?
// Извлекаем только 1 строку после пропуска.
