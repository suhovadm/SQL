Вычислить количество пациентов, у которых не указана дата рождения, не используя 
какие-либо условные конструкции (включая, но не ограничиваясь WHERE, HAVING, CASE, IIF)

SELECT COUNT(BDATE) - COUNT(*) as Number_Of_Patients_Without_BD
// COUNT(BDATE) - считаем количество ненулевых значений в столбце BDATE.
// COUNT(*) - посчитать общее количество строк в таблице CLIENTS.
// Вся конструкция - это разность между количеством ненулевых значений в BDATE и общим количеством строк в таблице.
Она дает количество клиентов, у которых отсутствует дата рождения.
// Результирующее значение сохраняется под псевдонимом Number_Of_Patients_Without_BD,
// оно и показывает количество клиентов у которых отсутствует дата рождения.

FROM CLIENTS;
// Говорим, откуда сделать выборку. Из таблицы CLIENTS.
